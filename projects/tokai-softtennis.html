01 / OVERVIEW

案件概要

東海大学ソフトテニス部の公式情報を、閲覧者ごとの目的に応じて整理したサイトです。 初めて見る人にはチーム紹介、関係者には投稿や試合情報、入部希望者には案内導線が すぐ届くような情報設計になっています。

SITE TYPE

大学部活動の公式Webサイト

部の紹介、活動情報、試合情報、入部案内を一つの導線の中で整理したサイトです。

AUDIENCE

新入生 / 保護者 / 関係者

初見ユーザーと継続閲覧ユーザーの両方が、必要情報にすぐアクセスできる構成です。

STRUCTURE

紹介・更新・募集の3軸

静的な紹介ページに加え、投稿や試合結果など更新系ページも含めた情報設計です。

PORTFOLIO VIEW
